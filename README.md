
## 快速开始

### 已支持

- Quark | 夸克网盘
- Uc | UC网盘
- Lanzou | 蓝奏云

### 依赖环境

- Python 3.7+
- Required dependencies (see [requirements.txt](./requirements.txt))  
  所需依赖请见 requirements.txt

### 安装

```bash
git clone https://github.com/your-username/your-repo-name.git
pip install -r requirements.txt
```
